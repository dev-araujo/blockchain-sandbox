## Como usar

### Clonar o repositório

```bash
git clone https://github.com/dev-araujo/blockchain-sandbox.git
cd blockchain-sandbox
```

### Executar um projeto

```bash
cd stacks-block-explorer
npm install
npm run dev
```

Acesse `http://localhost:3000` no navegador.
